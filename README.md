# Budget Planner

A simple budget planner, where you can track all your transactions in different categories, got some statistics and also can create monthly budgets.

[Demo](https://budget-planner.netlify.app/)

## Getting Started

### Installing

npm:

```
npm install
npm start
```

yarn:

```
yarn install
yarn start
```
